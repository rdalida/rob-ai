ChatGPT Codex

What is it?

ChatGPT Codex is an assistant that turns plain language tasks into working pull requests. It analyzes your repository, creates changes, then summarizes them for review.

Setup

  1. Clone the repository you want Codex to modify.
  2. Create and export your API key as OPENAI_API_KEY.
  3. Run the Codex command inside the project directory.

Usage

Describe your desired changes. Codex writes the code, commits it, and reports test results so you can merge with confidence.
